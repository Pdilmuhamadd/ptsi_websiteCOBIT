## üìå About The Project
**IT Governance Dashboard** adalah aplikasi berbasis web yang dikembangkan selama masa magang saya di **PT. Surveyor Indonesia (Juli - Agustus 2024)**.

Aplikasi ini dirancang untuk mendigitalisasi proses tata kelola IT menggunakan framework **COBIT 2019**. Tujuan utamanya adalah menggantikan proses manual dengan sistem otomatis yang lebih akurat, memberikan visibilitas yang lebih baik bagi pemangku kepentingan (stakeholders) terhadap kinerja proses IT, serta mendukung pengambilan keputusan berbasis data (*data-driven decision making*).

> **‚ö†Ô∏è Project Status:** This is an **MVP (Minimum Viable Product)** developed within a 2-month internship timeframe. The core assessment logic and database management are functional (~70% Completion), while the reporting module is currently in the prototype stage.

## Key Features
- **COBIT Framework Integration:** Implementasi struktur domain COBIT 2019 (APO, BAI, DSS, MEA) untuk menyelaraskan tujuan IT dengan tujuan bisnis.
- **Automated Maturity Calculation:** Algoritma otomatis untuk menghitung nilai kapabilitas (Capability Level) dan Gap Analysis tanpa rumus manual.
- **Dashboard Visualization:** Antarmuka (UI) yang *user-friendly* untuk menampilkan status tata kelola IT secara visual.
- **Efficient Data Management:** Sistem database MySQL yang terstruktur untuk penyimpanan dan pengambilan data audit yang cepat (CRUD).
- **PDF Reporting:** Fitur generate laporan hasil audit otomatis (Work in Progress).
